Question-1: What are the different ways to select an element in the DOM?

Here are the different ways to select elements in the DOM:
document.getElementById(): returns an element by its unique ID.
document.getElementsByClassName(): Selects elements by their class name & returns an HTMLCollection.
document.getElementsByTagName(): Selects elements by their tag name & returns an HTMLCollection.
document.querySelector(): Selects the first element that matches a specified CSS selector & returns an Element.
document.querySelectorAll(): Selects all elements that match a specified CSS selector & returns a NodeList.


Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: sets the HTML content of an element, including any child element.
innerText: sets the text content of an element, with some formatting and styling considerations and does not include any HTML child elements .
textContent: sets the all text content of an element, without any formatting or styling .


Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element instead of multiple child elements. When an event occurs on a child element of the parent, it "bubbles up" the DOM tree, allowing the parent's event listener to handle it. This approach improves performance by reducing the number of event listeners and simplifies event handling, especially for dynamic content or elements that are added or removed dynamically.


Question-4: What is event bubbling in the DOM?

Event bubbling is the order in which events are handled in the DOM. moves from the innermost element to the outermost element. When an event occurs on an element, it first triggers the event handlers on that element, then on its parent, then its grandparent, and so on, up the DOM tree. can be stopped using event.stopPropagation() method.


Question-5: How do you create, add, and remove elements using JavaScript?

Add:
document.createElement('elementName') - Creates a new element of the specified type.
parentElement.appendChild(newElement) - Adds the new element as the last child of the parent element.
Remove:
parentElement.removeChild(childElement) - Removes a specific child element from the parent element.
